# Supabase Tutorial (Postgres in the Cloud)

Supabase is basically “Postgres with a control panel and superpowers.” 

It’s great for teaching because students don’t need to install a DB server locally to get going.

## Part 1 - Setup
- Create a Supabase project
  - Go to Supabase and create a new project (students will need an account, you can use your GitHub account).
  - Choose an organization, name the project (ex: SDC###)
  - Set a strong DB password
  - Pick a region
  - Click on the Create New Project button
- Open the SQL Editor
  - In the project dashboard, click SQL Editor button.
  - Supabase explicitly supports creating tables either via the dashboard or straight SQL in the editor.
- Run a simple query to check your project
  - in the top, type ```select now();```
  - in the bottom half, click on run
  - If working properly, it should display the date and time
